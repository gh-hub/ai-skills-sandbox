# Glossary: thanks-claude

**Like** — a single thank-you record. Created every time a visitor clicks the like button. No identity or dedup attached; a visitor can create many.

**Story** — the optional free-text note a visitor can add to a like, describing what they loved about Claude and/or how it helped them.

**Hours saved** — an optional numeric field on a like, the number of hours the visitor estimates Claude saved them.
